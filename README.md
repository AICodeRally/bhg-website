# Blue Horizons Group Website

Next.js 16 marketing website for Blue Horizons Group, built with TypeScript, Tailwind CSS, and deployed on Vercel.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build & Deploy

```bash
npm run build
npm start
```

Deployed to Vercel at www.bluehorizonsgroup.com
